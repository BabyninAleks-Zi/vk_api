# Получите короткую ссылку

<img width="497" height="75" alt="image" src="https://github.com/user-attachments/assets/b2101b60-961b-4693-903a-c725931b7518" />

Это код для сотрудников международной компании "ООО Терминальная стадия".
При запуске кода в терминале через VK API Вы сможете получить короткую ссылку,
введя длинную ссылку более 25 символов, а так же получить информацию о том, 
сколько кликов по этой ссылке было произведено с момента её создания.

***

### Как установить

Прежде чем начать работу с репозиторием необходимо создать для репозитория 
виртуальное окружение и далее установить библиотеки из файла `requirements.txt`.

Для установки необходимых библеотек использующихся в репозитории, в консоли 
напишите команду и дождитесь установки всех необходимых для работы репозитория 
библиотек:

```
$ pip install -r requirements.txt
```

Документацию к библиотекам можете изучить на сайте 
[PyPI](https://pypi.org/).

***

### Как использовать

До начала работы с репозиторием, создайте в корневом каталоге файл `.env` и 
напишите переменную `VK_TOKEN` используя Сервисный Токен (Service Token) VK_API.

Для того чтобы узнать что делает этот скрипт и какие аргументы ему нужны, введите:

```
$ python main.py -h
```

Код работает просто, необходимо в консоли ввести команду:

```
$ python main.py 'YOUR LINK'
```

После ввода вы получите короткую ссылку, как на примере ниже:
